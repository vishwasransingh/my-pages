Links

  • HyperLinks: You can click on a link and jump to another document.
  • Local Links: Is a link to another page of the same website. Specified with a relative URL.

Target Atribute:

Specifies where to open the linked document. 
<a href="https://www.xyz.com/" target="_targeType">Visit xyz.com</a>
  • _blank - Opens in new tab or window.
  • _self - Default. Opens in same window.
  • _parent - Opens in the parent frame.
  • _top - Opens in the full body of the window
  • framename - Opens in a named frame.

Image as a link

Note : border:0; is added to prevent IE9 (and earlier) from displaying a border around the image (when the image is a link).

Link colors

  • Unvisited link: Blue
  • Visited link: Purple
  • Active link: Red
We can change default colors using CSS. 
        <style>
            a:link {
              color: green;
              background-color: transparent;
              text-decoration: none;
            }
            
            a:visited {
              color: pink;
              background-color: transparent;
              text-decoration: none;
            }
            
            a:hover {
              color: red;
              background-color: transparent;
              text-decoration: underline;
            }
            
            a:active {
              color: yellow;
              background-color: transparent;
              text-decoration: underline;
            }
            </style>

Bookmark Links

  • Usd to allow reder to jump to specific parts of a Web page.
  • Useful if the page is very long.
        <h2 id="C4">Chapter 4</h2>             ----Creating bookmark
        <a href="#C4">Jump to Chapter 4</a>    ----Adding link
