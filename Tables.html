Tables

Defining a table:

  • Table is defined with <table> tag.
  • Table row is defined with <tr> tag.
  • Table header is defined with the <th> tag.
  • Table data is defined with the <td> tag.
                <table style = "width:100%">
                    <tr>
                        <th>Firstname<th>
                        <th>lastname</th>
                        <th>Age</th>
                    </tr>
                    <tr>
                        <td>Satu</td>
                        <td>Ransing</td>
                        <td>25</td>
                    </tr>
                    <tr>
                        <td>Vishwas</td>
                        <td>Ransing</td>
                        <td>22</td>
                    </tr>
                </table<

Table-borders

Default: without borders. We need to set borders by using CSS border property. ex: 
                table, th, td{
                    border: 1px solid black;
                    border-collapse:collapse;
                }
'border-collapse:collapse' in CSS helps us to collapse borders into one border.

Other properties: 'padding: 1px', 'text-align', 'border-spacing:5px', etc.

'colspan' attribute is used to make cell span more than one column.

'rowspan' attribute is used to make a cell span in multiple rows.

<caption> tag adds capion to the table.

Table styling example:

                <style>
                    table {
                      width:100%;
                    }
                    table, th, td {
                      border: 1px solid black;
                      border-collapse: collapse;
                    }
                    th, td {
                      padding: 15px;
                      text-align: left;
                    }
                    table#t01 tr:nth-child(even) {
                      background-color: #eee;
                    }
                    table#t01 tr:nth-child(odd) {
                     background-color: #fff;
                    }
                    table#t01 th {
                      background-color: black;
                      color: white;
                    }
                    </style>
