Styling HTML with CSS

CSS describes how HTML elements are to be displayed on screen, paper or in other media. It can be added in 3 ways:

  1. Inline : By using style attribute in html elements. Used to apply a unique style to single element.
  2. Internal : By using style element in head section. Used to define a style for a single html page.
  3. External : by using an external CSS file.
  • CSS can control the layout of web pages all at once.
  • The most common way to add CSS is to keep styles in separate CSS files.

Internal CSS:

To be defined in style tag. ex: 
                style_tag
                h1 {
                    color: blue;
                    font-family: verdana;
                    font-size: 300%;
                  }
                  p  {
                    color: red;
                    font-family: courier;
                    font-size: 160%;
                  }
                  style_tag_end

The id Attribute

To define a specific style for one special element, add an id attribute to the element: 


                <p id="p01">I am different</p>

then define a style for the element with the specific id: 

                #p01 {
                    color: blue;
                }

The class attribue

To define a specific style for one special element.

                <p class="error">I am different</p>
            
                p.error{
                    color: red;
                }

External references:

External style sheets can be referenced with a full URL or with a path relative to the current web page.

Full URL to link to a style sheet:

        <link rel="stylesheet" href="https://www.xyz.come/html/styles.css">

Link to style sheet located in the html folder on the current website:

        <link rel="stylesheet" href="/html/styles.css">
