Quotation and Citation Elements

Short Quotations:

q: Browsers usually insert quotation marks around these. for example: Do not go gentle into that good night. Rage, rage against the dying of the light.The quote was italicised by using 'i' element whereas quotation marks came from 'q' element.

Block Quotes:

blockquote: Defines a section that is quoted from another source. Browsers usually indent the blockquote elements. Here's an example:

You can fool some people some all the time and you can fool all the people some of the times but you can't fool all the people all of the time.-Abraham Lincoln

Abbreviations:

abbr: Defines abbreviation or acronym. Using them can give useful info to browsers, translation systems and search engines. Example:

TheWHO was founded in 1948.

Cite for Work Title:

cite: Defines title of a work. browsers usually display it in italics. Example:

A man standing by the car by Some Other Man. Captured in 2019.

Bi-Directional Override:

bdo: Used to override the current text direction. dir="rtl": Right to Left. dir="ltl": Left to right. Example:

This text will be written from the right to left.

Address for Contact Information:

address: Defines contact info of a doc or an article. It is usually displayed in Italics. Most browsers add a line break before and after the element. Example:

Written by Vishwas ransing
Visit us at: imaginarywebsite.com
Walki Road, Wadgaon Tandali, Ahmednagar
Maharashtra.
