Image Maps

With image maps, we can add clickable areas. <map> tag defines an image map.
Clickable Areas are defined using certain elements:
  • map : Used to create an image map. is linked to image using name attribute.
  • area : Used to define the clickable area.
  • shape : Defines the shape of clickable area. Can be rect, circle, poly, default (entire region).
  • coords : Defines coordinates.
The clickable area can also be used to trigger a JavaScript function.
example: 
            <map>
            <img src="img_name.jpg" alt="alt_text" usemap="#map_name">
            <map name="map_name">
            <area shape="rect" coords="34,44,270,350" alt="click_1_alt" href="computer.htm">
            <area shape="rect" coords="290,172,333,250" alt="click_2_alt" href="phone.htm">
            <area shape="circle" coords="337,300,44" alt="click_3_alt" href="coffee.htm">
            </map>

Background images

Can be specified for any element as well as for whole page (body element). 
            <div style="background-image: url('img_name.jpg');">
Other examples: 
            <style>
                body {
                  background-image: url('image_name.jpg');
                  background-repeat: no-repeat;====> to avoid the background image from repeating
                  background-attachment: fixed;====> to make sure that image covers te entire element 
                  background-size: cover;==========> to cover entire element with no stretching
                  background-size: 100% 100%;======> to stretch the image to match the area
                }
            </style>

Picture element

Allows us to display different pictures for different devices or screen sizes.

The <picture> element contains a number of <source> elements, each referring to different image sources. This way the browser can choose the image that best fits the current view and/or device.

Each <source> element have attributes describing when their image is the most suitable. 

            <picture>
            <source media="(min-width: 650px)" srcset="img_food.jpg">
            <source media="(min-width: 465px)" srcset="img_car.jpg">
            <img src="img_girl.jpg">
            </picture>

Note:It is recommended to specify an <img> element as the last child element of the picture element so that the browsers which dont support picture element could use them.
