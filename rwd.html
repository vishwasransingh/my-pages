Responsive web design

It is about using html and css to automatically resize, hide, shrink or enlarge a website to make it look good on all the devices with different screen sizes.

Viewport

Syntax: <meta name="viewport" content="width=device-width, initial scale=1.0">

Responsive images

If the CSS width property is se to 100%, the image will be responsive and scale up and down.

Syntax: <img src="imgname.jpg" syle="width:100%;">

If the max-width property is set to 100%, the image will scale down if it has to, but will never scale up than its original size.

picture element allows you to define different images for different browser window sizes.

            <picture>
                <source srcset="img_smallflower.jpg" media="(max-width: 600px)">
                <source srcset="img_flowers.jpg" media="(max-width: 1500px)">
                <source srcset="flowers.jpg">
                <img src="img_flowers.jpg" alt="Flowers" style="width:auto;">
            </picture>

Responsive text

Text sizes can be set with a "vw" (viewport width) unit.

Tha way the text size will follow the size of the browser window.

Syntax:
<h1 style="font-size:10vw">Hello World</h1>
